# 設計

## 画面(入出力の例)

python movie.py <サブコマンド> [引数]

### サブコマンド

- add <タイトル> <ジャンル> <評価>    映画を追加
- list                                映画一覧を表示
- search <タイトル>                   映画を検索
- edit <タイトル> <評価>             映画の評価を変更
- delete <タイトル>                   映画を削除
- sort                                評価順に表示

### 実行サンプル

python movie.py add タイタニック 恋愛 5
追加しました

python movie.py list

タイトル,ジャンル,評価
タイタニック,恋愛,5
ターミネーター2,SF,5

## データ

### movie.csv

映画情報をCSV形式で保存する。

title,genre,rate
タイタニック,恋愛,5
ターミネーター2,SF,5

## 処理の流れ

### add

1. タイトル、ジャンル、評価を受け取る
2. movie.csvを読み込む
3. 同じタイトルが存在するか確認する
4. 存在しなければ追加する
5. movie.csvへ保存する

### list

1. movie.csvを読み込む
2. 登録されている映画を表示する

### search

1. タイトルを受け取る
2. movie.csvを検索する
3. 一致する映画を表示する

### edit

1. タイトルと新しい評価を受け取る
2. movie.csvを検索する
3. 評価を更新する
4. 保存する

### delete

1. タイトルを受け取る
2. movie.csvから削除する
3. 保存する

### sort

1. movie.csvを読み込む
2. 評価順に並び替える
3. 表示する

## 構成

- movie.py      実装本体
- movie.csv     映画データ

文字コード：UTF-8
ライブラリ：Python標準ライブラリのみ
