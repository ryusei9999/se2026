# 映画管理CLI

見た映画を登録し、評価や感想を管理するCLIソフト。

## 機能要件

1. 映画を追加する
   - 映画タイトルを登録できる
   - ジャンルを登録できる
   - 評価（1～5）を登録できる

2. 登録済み映画を編集する
   - タイトルを変更できる
   - 評価を変更できる

3.映画を削除する

4. 映画一覧を表示する

5. 映画名で検索する

6. 評価順に並び替える

## 非機能要件

1. CLIで動く

2. 内部データはCSVで保存

3. 日本語が文字化けしない（UTF-8）

4. Python標準ライブラリだけで動く

5. 各コマンドの処理は1秒以内
