# wallet - 簡易コマンドライン家計簿

`wallet` は、ターミナルから1行のコマンドで支出をサクサク記録し、今月の予算管理ができるシンプルなPython製・家計簿ツールです。
アプリを開く手間のすら惜しい、エンジニアやキーボード派のための実用的なツールです。

##  主な機能

- **爆速の支出記録**：金額、内容、カテゴリで記録。
- **予算管理と警告**：今月の予算を設定し、使用率や残高を自動計算。予算オーバー時は警告を表示。
- **当月自動集計**：過去の履歴を残しつつ、`summary` では自動で「今月」のデータのみを抽出・集計。
- **カテゴリ内訳**：何に一番お金を使っているかをランキング形式で可視化。
- **履歴管理**：ID指定によるピンポイントのデータ削除機能。
