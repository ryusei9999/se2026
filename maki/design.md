# 設計

## 画面(入出力の例)
python wallet.py <サブコマンド> [引数]

### サブコマンド：
- budget <金額>             今月の予算を設定・更新
- buy <金額> <内容> [カテゴリ] 支出を追加
- rm <ID>                  指定したIDの支出を削除
- list                     今月の支出一覧を表示
- summary                  今月の統計と予算残高を表示
- history                  過去すべての履歴を表示

### 実行サンプル：
```
$ python wallet.py buy 1200 ランチ
カテゴリが指定されていません。
カテゴリを入力してください (例: 食費, 交際費): 食費
追加した（本日合計: 1,200円）

$ python wallet.py summary
 === 【2026-05】 今月の収支サマリー ===
    設定予算 : 50,000 円
    支出合計 : 1,200 円
    予算残り : 48,800 円
```
## データ
### config.csv
予算などの設定データをKey-Value形式で保存する。
```csv
key,value
budget,50000
```

### wallet.csv
支出データを保存する。IDは削除しても詰めず、常に自動インクリメント（最大ID + 1）する。
```
id,date,amount,description,category
1,2026-05-27,1200,ランチ,食費
2,2026-05-27,8500,飲み会,交際費
```


## 処理の流れ


### budget
引数から金額（文字列）を取得する

金額が数値に変換できない場合は、エラーで止める

config.csvに budget,金額 を書き込んで更新する

### buy
引数から金額と内容を取る。金額が数値でなければエラーで止める

カテゴリの指定が無ければ、対話で聞く（空欄なら「未分類」とする）

wallet.csvを読む

現在の最大IDを調べ、新ID = 最大ID + 1 で採番する（データがなければ1）

現在の日付（YYYY-MM-DD）を取得する

データを末尾に追加して書き戻す

画面に「本日合計: 〇〇円」を計算して表示する

### list

### summary

### history
wallet.csvをすべて読み込む

登録されている過去すべての履歴を一覧表示する
