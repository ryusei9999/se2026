# 要求仕様書

## 1. ソフトウェア概要
本ソフトウェアは、日々のタスク管理を簡単に行える「スマートタスク管理アプリ」である。  
ユーザーはタスクの登録・編集・完了管理を直感的に行うことができ、日常の予定管理を効率化する。

---

## 2. 機能要件（5つ）
1. タスクを新規作成できる  
2. タスクの編集・削除ができる  
3. タスクに締切日・優先度を設定できる  
4. タスクをカテゴリごとに分類して表示できる  
5. 完了済みタスクを一覧で確認できる  

---

## 3. 非機能要件（5つ）
1. アプリ起動後 3 秒以内にホーム画面を表示する  
2. 直感的に操作できる UI を提供する  
3. データはローカルに自動保存され、アプリ終了後も保持される  
4. 1,000 件以上のタスクが登録されても動作が重くならない  
5. ユーザーの誤操作を防ぐため、削除時には確認ダイアログを表示する  

---

## 4. 画面一覧（3つ）

### 4.1 ホーム画面
- 登録済みタスクの一覧を表示  
- タスクの追加ボタンを配置  
- カテゴリ別フィルタを利用可能  

### 4.2 タスク作成画面
- タスク名、締切日、優先度、カテゴリを入力  
- 保存ボタンでタスクを登録  

### 4.3 設定画面
- テーマ変更（ライト/ダーク）  
- 通知設定のオン/オフ  
- データバックアップ設定
